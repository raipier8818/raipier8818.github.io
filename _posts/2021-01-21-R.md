<h2>1. 변수</h2>

 - 변수의 자료형은 저장하지 않는다
 - 이름은 문자나 .(dot)으로 시작한다
 - 대소문자를 구분한다


```R
A <- 10     # A에 10을 대입
B <- A+20   # B에 A+20을 대입
```

<br>
<h4>사용가능한 자료형</h4>

 - 숫자 : 1, 23, 0.123, -11.1
 - 문자 : "A", "Rstudio"
 - 논리형 : TRUE, FALSE, T, F
 - 특수한 값
    - NULL : 비어있는 값
    - NA : 결측값
    - NaN : 수학적으로 정의가 불가능한 값
    - Inf, -Inf : 양의 무한대, 음의 무한대
    
<br>

<h2>2. 벡터</h2>

 - 벡터 생성

```R
x <- c(1,2,3)                                # 정수형 벡터
y <- c("a","b","c")                          # 문자형 벡터
z <- c(TRUE,FALSE,TRUE,TRUE)                 # 논리형 벡터

v1 <- 50:90  # 50부터 90까지
v2 <- c(1,2,3, 50:90)                        # 1,2,3,50,51,...,90
v3 <- seq(1,101,3)                           # seq(시작,종료,간격)

v4 <- rep(1,times=5)                         # req(값, 반복횟수)
v5 <- rep(c("a","b","c"), each=3)            # each : 원소 반복
v7 <- rep(c("a","b","c"), times=3)           # times : 벡터 반복
```

<summary>함수</summary>
<div markdown = "1">

- sum(vector) : 자료의 합
<br>
- mean(vector) : 자료의 평균
<br>
- max(vector), min(vector) : 자료의 최댓값, 최솟값
<br>
- var(vector) : 자료의 분산
<br>
- sd(vector) : 자료의 표준편차
<br>
- sort(vector) : 자료 정렬
<br>
- sort(vector, bool) : 자료 정렬, bool == true == 내림차순 정렬
<br>
- range(vector) : 자료의 범위 (최댓값 ~ 최솟값)
<br>
- length(vector) : 자료의 길이(개수)
<br>

</div>
